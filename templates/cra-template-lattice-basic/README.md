## How to use

### With CRA

`npx create-react-app my-app --template cra-template-lattice-basic`

### With clap

`clap myApp`

_Note: `basic` is the default Lattice example_

## Features

- Similar to CRA.
- Create your upcoming Lattice App with one command!
